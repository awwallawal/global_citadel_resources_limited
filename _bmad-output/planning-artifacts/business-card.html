How to print — Single Card PDF
  1. Press Ctrl + P (or Cmd + P on Mac)
  2. Destination: Save as PDF
  3. Paper size: leave as default (browser picks 85 × 55 mm from CSS)
  4. Margins: None
  5. Scale: Default or Custom 100% (never “Fit to page”)
  6. Background graphics: ON ← critical!

Result: 2-page PDF (front + back), each page exactly business-card size with rounded corners.

How to print — A4 Print Sheet
  1. Press Ctrl + P (or Cmd + P on Mac)
  2. Destination: Save as PDF
  3. Paper size: A4
  4. Layout: Portrait
  5. Margins: None
  6. Scale: Default or Custom 100%
  7. Background graphics: ON ← critical!

Result: 2-page A4 PDF — page 1 has 8 fronts, page 2 has 8 backs. 3 mm bleed + crop marks included. Hand this file directly to your print shop.
